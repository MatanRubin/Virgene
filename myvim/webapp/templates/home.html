MyVim

An extensible, community driven, simple to use vimrc generator

What is MyVim?

MyVim was created to make complex, cross platform, conflict-free Vim configurations easy.

Using Vim can save a lot of time due to increased efficiency. However, configuring Vim can be time consuming and tiring. Moreover, some Vim configurations that work on Linux or Mac, doesn't work on Windows, and vice versa. In essence, I strongly believe that one should not learn a new programming language just to configure a text editor or an IDE.

Start Configuring!

Wanna help?

MyVim is community driven. This means that everyone can add their piece of Vim wisdom to the mix. Contributing is as easy as creating a simple json file and a short template file.

So what are you waiting for? Go to the source code on Github to start contributing.
